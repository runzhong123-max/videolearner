你正在为 VideoLearner 生成结构化学习笔记。

输入：
- 用户意图：{user_prompt}
- 学习上下文：{context_text}
- 当前启用的输出键：{json_keys}

硬性输出要求：
1. Return ONLY JSON.
2. 不要输出 Markdown。
3. 不要输出代码块标记（如 ```）。
4. 不要输出 JSON 之外的任何解释、前后缀、注释。
5. 只输出一个 JSON 对象。

JSON 结构要求：
- 必须包含以下键（且值必须是字符串）：
  - summary
  - expansion
  - inspirations
  - guidance
- 若某项无内容，返回空字符串 ""。
- 不要返回 null、数组、对象作为上述字段值。
- 如 {json_keys} 中包含其他可选键，可一并返回，值同样使用字符串；无内容则返回 ""。

字段语义：
- summary：总结本次学习最核心的内容、关键概念与主线。
- expansion：给出进一步学习方向、相关概念或延展建议。
- inspirations：提炼用户灵感/想法；若无灵感信息则返回 ""。
- guidance：给出下一步可执行学习动作（简洁、具体）。

质量要求：
- 严格基于上下文，不编造。
- 如信息不足，可在对应字段中简要说明不足点。
- 内容应简洁、教学导向、可执行。
- 除特殊情况外，字段内容请使用中文。
